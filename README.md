# Personal Site

Astro + TypeScript + native CSS personal website for Kanshao 2077.

## Commands

```bash
npm install
npm run dev
npm run build
```

## Content Rules

- Feishu is the draft source, but website copy lives in `src/data/site.ts`.
- Large videos stay on external platforms.
- Real images and covers go in `public/covers/`, `public/images/`, and `public/avatar/`.
- The first version keeps five pages only: Home, Works, About, Gallery, Contact.
