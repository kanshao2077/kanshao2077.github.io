# Personal Website Design Brief

## Direction

Retro-futuristic, editorial, refined. The site should feel like an AI product operator's portfolio, not a template resume site.

## Visual Keywords

- Refined retro future
- Soft but not weak
- Editorial typography
- Light technical texture
- AIGC work as the visual center

## Palette

- Ink: `#171512`
- Paper: `#f7f5ef`
- Mist: `#e7ece8`
- Teal: `#0f9f9a`
- Vermilion: `#d75f45`
- Cobalt: `#315f9f`
- Brass: `#b78b35`

## Typography

- Display: Instrument Serif / Georgia fallback
- Body: IBM Plex Sans / system sans fallback

## Components

- Cards use sharp 6px radius.
- Works cards carry proof text, not only titles.
- Mobile is one-column first.
- Motion is limited to reveal, hover, and filter transitions.

## Do Not Use

- Purple-blue AI gradients
- Neon cyberpunk overload
- Template resume blocks
- Full-screen decorative effects
- Auto-playing music
- Huge videos in the repository
